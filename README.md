# 睡前内容生成器 🌙

一个基于Streamlit和通义千问API的睡眠辅助应用，提供三个核心功能帮助用户改善睡眠。

## 功能介绍

### 📖 睡前故事生成
- 用户选择主题（森林/星空/海洋等）
- AI创作原创睡前故事
- 可调节故事长度和风格

### 🧘 冥想引导脚本
- 根据用户当前情绪生成个性化冥想引导
- 支持多种冥想类型（呼吸冥想、身体扫描、正念冥想等）
- 可选择5-20分钟的冥想时长

### 📝 睡眠日记模板
- 根据用户今日状态生成个性化日记模板
- 帮助用户记录睡眠情况和感受
- 提供睡前建议和正向引导

## 技术栈

- **前端框架**：Streamlit
- **AI模型**：通义千问 API (qwen-turbo)
- **开发语言**：Python 3.10+

## 本地运行

### 1. 安装依赖
```bash
pip install streamlit dashscope
```

### 2. 配置API Key

在 `sleep_app.py` 第13行，将 `your-api-key-here` 替换为你的通义千问API Key。

### 3. 运行应用
```bash
streamlit run sleep_app.py
```
